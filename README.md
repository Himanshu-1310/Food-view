#  Food View – Food Discovery & Reels Web Application

##  Overview

**Food View** is a full-stack MERN web application that allows users to explore food content through a reels-style interface. Users can browse food items, watch short food videos, and save their favorite content, while food partners can manage and upload their listings.

---

## 🛠 Tech Stack

**Frontend**

* React.js
* Tailwind CSS
* React Router

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB (Mongoose)

**Authentication**

* JSON Web Tokens (JWT)

**Media Handling**

* Multer
* ImageKit

---

## ✨ Key Features

### 👤 User Features

* Browse food items and reels
* Scroll-based content experience (like reels)
* Save favorite food content
* Secure login & signup (JWT authentication)

### 🧑‍🍳 Food Partner Features

* Add and manage food listings
* Upload images and videos
* Manage their content through dashboard

### ⚙️ Backend Features

* RESTful APIs for users, food partners, and food items
* Secure authentication & authorization using JWT
* Efficient data handling with MongoDB
